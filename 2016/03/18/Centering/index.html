Horizontal Centering

Case 1:
Child element of ‘display: inline-block’
we can make the parent ‘text-align: center;’

However, notice that this technique will only work if the parent is a block element. Which means, if the parent is using display: flex, this technique will not work.

Case 2:
Child element of ‘display: block’
we can use traditional technique, margin: 0 auto.
