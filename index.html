How to elegantly make a header

2016-03-20
  • web development
  • css

So, you want to make a header. 这里有一个一步到位的解决方案.
Take this as an example.

We should first make a container for this header

And we use header tag here for semantics. Then we assign a height to the header, and make it flex display. Then we can use flexbox’s special skill, align-items: center to align everything in the header vertically centered. We also want to add some padding in the left and right hand side of the header.

Now we can arrange our things. First we put our logo, and then we put a span of class separator, then we put other things. Because our header is of display:flex, our separator class, with attribute flex:1, will automatically adjust the distance between things on the left and things on the right.

For other things, like the log-in stuffs, the only thing we need to do is put them under the header, set the font size, style, and margin, and then we are done.
